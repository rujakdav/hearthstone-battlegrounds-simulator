# Battlegrounds simulator
This projects 
is an implementation of Hearthstone Battlegrounds in C++ with the use of Cython.

## Directory Structure

- `build`: Directory containing build files. 
- `cpp_library`: Directory containing C++ source code for the project.
- `docs`: This directory holds documentation files generated by Doxygen.

- `src`: The main source code directory.

  - `generated`: This subdirectory contains generated .pyd files.

  - `wrappers`: Cython .pyx files that wrap C++ code.


- `Doxyfile`: Configuration file for Doxygen.


## Getting Started

Run the following commands to build the .pyd files:

```bash
python src/setup.py build_ext --inplace
```

then run main.py to start the program.

```bash
python src/main.py
``` 
## Documentation

To generate documentation, run the following command:

```bash
doxygen Doxyfile
```
